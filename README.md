# 💰 Lista de Despesas

Projeto simples desenvolvido com **HTML, CSS e JavaScript**, com foco na prática de **manipulação do DOM**, eventos e atualização dinâmica da interface.

## 📋 Sobre o projeto

A aplicação permite cadastrar despesas informando:

- Nome da despesa
- Categoria
- Valor da despesa

As despesas adicionadas são exibidas em uma lista dinâmica, com atualização automática da quantidade de itens e do valor total cadastrado.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 Conceitos praticados

- Seleção de elementos do DOM
- Manipulação de elementos HTML
- Eventos de formulário
- Criação dinâmica de componentes
- Atualização de conteúdo em tempo real
- Formatação de valores monetários
- Organização de código JavaScript

## 📸 Preview

Interface para cadastro e gerenciamento simples de despesas:

- Cadastro de despesas
- Seleção de categoria
- Controle do total gasto
- Lista dinâmica de despesas

## 📚 Objetivo

Este projeto foi desenvolvido com finalidade de estudo e prática de JavaScript, especialmente para reforçar conceitos de manipulação do DOM e interação com o usuário.
